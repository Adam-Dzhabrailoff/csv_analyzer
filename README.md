# csv_analyzer
Данная утилита генерирует отсчёт на основе входных csv файлов.

## Описание
Скрипт читает один или несколько csv файлов, после чего производит анализ данных на основе стратегии, по итогу которого генерируется таблица данных.
Стратегия, в соответствии с которой производится анализ данных в csv файлах, определяется параметром --report. После обработки данных, скрипт отображает результат в формате таблицы в терминале пользователя.

## Доступные типы отчёта
* report - среднее арифмитическое производительности сотрудников в конкнертной позиции.

## Параметры скрипта
| Опция| Описание|
|--------------|-------------------------------------------------------|
| '--file'| Путь или пути к csv-файлу или csv-файлам|
| '--report'| Тип отчёта: 'report'|
